BloomAPI

We are currently building the first version of BloomAPI. BloomAPI will be a self-sustaining API for the National Provider Identifier (NPI) data from the CMS NPI Data Dissemination. It is motivated by the current lack of a free (libre/ as in free speech), non-stale, API for this datasource. If you'd like to contribute, checkout the github project.
